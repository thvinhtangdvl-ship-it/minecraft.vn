# 🎮 Minecraft.vn - Dự án Server Minecraft GitHub

Chào mừng bạn đến với repository **minecraft.vn**! Đây là nơi lưu trữ và phát triển server Minecraft chạy trên nền tảng GitHub.

## 🚀 Trạng thái dự án
- **Phiên bản Game:** 1.20.1 (hoặc bản bạn muốn)
- **Loại Server:** Paper / Spigot / Fabric
- **Chế độ:** Survival / Skyblock

## 🛠 Cách tham gia / Chạy thử
1. Mở **Codespaces** từ nút `<> Code`.
2. Chạy lệnh `bash start.sh` để khởi động server.
3. Sử dụng địa chỉ IP từ công cụ tunnel (ngrok/playit) để kết nối.

## 📂 Cấu trúc thư mục
- `/world`: Chứa dữ liệu bản đồ.
- `/plugins`: Chứa các tính năng mở rộng.
- `server.jar`: File nhân server chính.

## 🤝 Đóng góp
Nếu bạn muốn giúp đỡ phát triển server, hãy nhấn **Fork** và tạo **Pull Request** nhé!
